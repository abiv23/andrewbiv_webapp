RetroPie: Raspberry Pi Project

Originally gifts for my Groomsmen, the Raspberry Pi is a powerful and fun machine. Next up is a wireless storage device.

Project Details

The case was 3d printed, the motherboard is a Raspberry Pi 3b running Raspian and Emulation Station. Storage is a 32gb micro SD card, storing over 3,000 roms on various systems such as NES, SNES, SEGA Genesis, Nintendo64 and more.
